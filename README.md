# flutter_wanandroid
## 一 项目介绍
* 本项目基于：GetX开发
* 本项目所使用的API均来自：WanAndroid官网(https://www.wanandroid.com/blog/show/2)


## 二 开发工具及版本
* 开发系统：macos 12.3.1
* 开发工具：Intellij IDEA Community 2021.1
* Flutter：3.0.0
* Dart: 2.17.0
* .gitignore：使用的Dart类型
